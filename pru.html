Automatiza tu WhatsApp con estilo

Rin Itoshi es un bot modular para WhatsApp con comandos de música, utilidades, moderación y herramientas multimedia. Fácil de instalar y personalizar.

Descarga audio desde YouTube y reproductores; optimizado para archivos pesados.

API

Integraciones

Fácil de conectar con APIs externas para funciones extra (OCR, TTS, IA).

PM2

Autorestart

Uso recomendado con PM2 para mantener el bot corriendo en segundo plano.

Estado

Script preparado — guarda y edita .env con tus credenciales.
Instalación básica 
git clone https://github.com/Yuji-XDev/Rin-Itoshi-Bot
cd Rin-Itoshi-Bot
pnpm install
cp .env.example .env
# editar .env y luego:
pnpm start
Termux (nota)
Si usas Termux instala: pkg install python clang make pkg-config y usa pnpm.
